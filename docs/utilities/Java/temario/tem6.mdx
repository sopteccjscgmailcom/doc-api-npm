# 🔥 Azure Databricks

**Azure Databricks** es un servicio de **análisis de datos y machine learning en la nube**, basado en **Apache Spark**, optimizado para **procesamiento masivo de datos (Big Data), inteligencia artificial y análisis en tiempo real**.  

🚀 **Combina la potencia de Apache Spark con la facilidad de uso de Azure** para procesar grandes volúmenes de datos de manera rápida y escalable.  

---

## 🛠 **¿Para qué se usa Azure Databricks?**  

✅ **Procesamiento de Big Data** → Analizar grandes volúmenes de datos estructurados y no estructurados.  
✅ **Machine Learning (ML)** → Entrenar modelos de IA con bibliotecas como **MLflow, TensorFlow, PyTorch, Scikit-Learn**.  
✅ **Análisis en Tiempo Real** → Procesar flujos de datos con **Spark Streaming**.  
✅ **Integración con Azure** → Se conecta con **Azure Data Lake, SQL Database, Synapse Analytics, Power BI, etc.**  
✅ **ETL (Extract, Transform, Load)** → Transformar datos antes de almacenarlos en bases de datos o data lakes.  

---

## 🔄 **¿Cómo Funciona Azure Databricks?**  

1️⃣ **Crea un Workspace** en Azure.  
2️⃣ **Define Clusters** (máquinas virtuales con Spark para procesar datos).  
3️⃣ **Desarrolla Notebooks** en **Python, Scala, SQL o R**.  
4️⃣ **Procesa Datos** desde **Azure Data Lake, Blob Storage, CosmosDB, SQL Server, etc.**.  
5️⃣ **Analiza y Visualiza** con **Power BI o MLflow**.  

---

## 📌 **Ejemplo de Código en Azure Databricks (Python + Spark)**  

🔹 **Cargar datos desde Azure Data Lake y procesarlos con Spark**  

```python
from pyspark.sql import SparkSession

# Crear sesión de Spark
spark = SparkSession.builder.appName("EjemploDatabricks").getOrCreate()

# Leer datos desde Data Lake
df = spark.read.csv("abfss://container@datalake.dfs.core.windows.net/datos.csv", header=True, inferSchema=True)

# Filtrar y mostrar datos
df_filtered = df.filter(df["ventas"] > 1000)
df_filtered.show()
```
