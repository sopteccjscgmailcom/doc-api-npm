# ☁ Microsoft Azure

**Azure** es la plataforma de **computación en la nube** de Microsoft que ofrece **infraestructura, plataformas y servicios** para ejecutar aplicaciones, almacenar datos y gestionar recursos de TI sin necesidad de servidores físicos propios.  

---

## 🔹 **¿Qué puedes hacer con Azure?**  

✅ **Máquinas Virtuales (VMs)** → Correr servidores en la nube sin comprar hardware.  
✅ **Bases de Datos** → SQL Server, MySQL, CosmosDB, entre otras.  
✅ **Almacenamiento** → Guardar archivos, copias de seguridad y datos masivos.  
✅ **Despliegue de Aplicaciones** → Hospedar apps web, APIs y microservicios con **Azure App Service**.  
✅ **Inteligencia Artificial (IA)** → Servicios de machine learning, reconocimiento de voz e imágenes.  
✅ **DevOps y CI/CD** → Integración continua con **Azure DevOps y GitHub Actions**.  
✅ **Seguridad y Redes** → Firewalls, VPNs y protección de datos con **Azure Security Center**.
