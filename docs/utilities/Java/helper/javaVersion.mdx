# 📊 Características nuevas segun versiones
---

### **Java 8 (2014) - 🔥 Revolución Funcional**  
✅ **Lambdas** → `(x) -> x * 2` 🏎️ (Código más conciso)  
✅ **Streams API** → `lista.stream().filter(x -> x > 10).collect()` 🚀 (Procesamiento eficiente de colecciones)  
✅ **`Optional<T>`** → Evita `NullPointerException` 🛑  
✅ **Default Methods en interfaces** → `default void metodo() {}` 🔄 (Permite agregar métodos sin romper compatibilidad)  
✅ **Nueva API de Fechas** (`java.time`) → Adiós `java.util.Date` ⏳  

---

### **Java 11 (2018) - ⚡ Más simple y moderno**  
✅ **`var` en variables locales** → `var mensaje = "Hola";` 🔍 (Menos código)  
✅ **Ejecución directa de archivos `.java`** → `java MiArchivo.java` 🏹 (Sin compilar antes)  
✅ **Nuevo cliente HTTP** → `HttpClient.send(request, BodyHandlers.ofString())` 🌍 (Simplifica peticiones web)  
✅ **Eliminación de Java EE y CORBA** → 🔥 (`javax.xml.bind`, `javax.activation`, etc.)  
✅ **ZGC y Epsilon GC** → 🔄 Mejoras en el Garbage Collector  

---

### **Java 17 (2021) - 🚀 Más rápido y seguro**  
✅ **Pattern Matching en `switch`** → `case String s -> System.out.println(s.length());` 🌀 (Menos `instanceof`)  
✅ **Sellado de clases (`sealed`)** → Controla qué clases pueden extender (`permits`) 🔒  
✅ **`Records` mejorados** → `record Persona(String nombre, int edad) {}` ✨ (Menos código para POJOs)  
✅ **Mejoras en rendimiento y seguridad** 🔥  

---

### **Java 21 (2023) - 🦾 Última LTS con lo mejor de lo mejor**  
✅ **`Virtual Threads` (Loom)** → 🚀 Hilos ligeros para mayor escalabilidad (`Thread.ofVirtual().start()`)  
✅ **`Scoped Values`** → 🧵 Mejor gestión de datos en hilos  
✅ **`Sequenced Collections`** → 📜 Nueva interfaz para colecciones con orden garantizado  
✅ **`Record Patterns`** → 📦 Más poder en desestructuración de objetos  
✅ **`Pattern Matching para `switch` estable** → 🔄 (Adiós al `instanceof` en cascada)  
✅ **Mejoras en el rendimiento del GC y JIT** 💨  

---

📌 **Resumen en una línea:**  
🔹 **Java 8**: Lambdas, Streams y `Optional` 🏎️  
🔹 **Java 11**: `var`, HTTP Client y mejor GC ⚡  
🔹 **Java 17**: `switch` mejorado, `records` y sellado de clases 🚀  
🔹 **Java 21**: Virtual Threads, Scoped Values y Collections avanzadas 🦾
