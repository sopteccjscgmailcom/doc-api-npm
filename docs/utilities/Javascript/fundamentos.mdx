# JavaScript
## 🧩 **Introducción**

### 📌 ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se usa principalmente para darle **interactividad** a las páginas web 🌐. Se ejecuta en el navegador.

### 📄 Primer script

```html
<!DOCTYPE html>
<html>
<head>
  <title>Mi primer JS</title>
</head>
<body>
  <h1>Hola Mundo</h1>
  <script>
    alert("¡Hola desde JavaScript!");
  </script>
</body>
</html>
```

## 🧮 **Variables y Tipos de Datos**

### ✍️ Declarar variables

```js
let nombre = "Juan";     // variable que puede cambiar
const edad = 30;         // constante que no cambia
var ciudad = "Lima";     // forma antigua (no recomendada)
```

### 📚 Tipos de datos

* String (texto): `"Hola"`
* Number (número): `42`
* Boolean (booleano): `true` o `false`
* Null: `null`
* Undefined: `undefined`
* Object y Array (más adelante)

## 🔁 **Operadores y Condicionales**

### ➕ Operadores básicos

```js
let x = 10 + 5;  // Suma
let y = 10 - 2;  // Resta
let z = 4 * 3;   // Multiplicación
let w = 8 / 2;   // División
```

### 🤔 Condicionales

```js
let edad = 18;
if (edad >= 18) {
  console.log("Eres mayor de edad");
} else {
  console.log("Eres menor de edad");
}
```

## 🔁 **Ciclos**

### 🔄 For

```js
for (let i = 0; i < 5; i++) {
  console.log("Repetición número " + i);
}
```

### 🔁 While

```js
let i = 0;
while (i < 5) {
  console.log("Valor de i: " + i);
  i++;
}
```

## 🧠 Módulo 1: Funciones Avanzadas

### 🔁 Funciones de orden superior

Son funciones que **reciben o retornan otras funciones**.

```js
function saludar(nombre) {
  return "Hola " + nombre;
}

function procesarEntradaUsuario(callback) {
  let nombre = "Luis";
  console.log(callback(nombre));
}

procesarEntradaUsuario(saludar); // Hola Luis
```

### 🔄 Métodos para arrays: `.map()`, `.filter()`, `.reduce()`

```js
let numeros = [1, 2, 3, 4];

// map
let dobles = numeros.map(num => num * 2); // [2, 4, 6, 8]

// filter
let pares = numeros.filter(num => num % 2 === 0); // [2, 4]

// reduce
let suma = numeros.reduce((acc, num) => acc + num, 0); // 10
```

## 🧱 Módulo 2: Destructuración y Spread

### 📦 Destructuring

```js
let persona = { nombre: "Ana", edad: 30 };
let { nombre, edad } = persona;

console.log(nombre); // Ana
```

### 🌪️ Spread y Rest

```js
let frutas1 = ["🍎", "🍌"];
let frutas2 = ["🍇", ...frutas1]; // ["🍇", "🍎", "🍌"]

function sumar(...numeros) {
  return numeros.reduce((a, b) => a + b);
}
```

## ⏳ Módulo 3: Asincronía

### 🕒 Callbacks

```js
function cargarDatos(callback) {
  setTimeout(() => {
    console.log("Datos cargados");
    callback();
  }, 2000);
}

cargarDatos(() => console.log("Después de cargar"));
```

### 🌐 Promesas

```js
let promesa = new Promise((resolve, reject) => {
  let exito = true;
  if (exito) resolve("Todo OK");
  else reject("Algo falló");
});

promesa.then(res => console.log(res)).catch(err => console.log(err));
```

### ⚡ Async/Await

```js
async function obtenerDatos() {
  try {
    let respuesta = await fetch("https://jsonplaceholder.typicode.com/posts/1");
    let datos = await respuesta.json();
    console.log(datos);
  } catch (error) {
    console.log("Error:", error);
  }
}
```

## 📦 Módulo 5: Módulos y Exportaciones

### ➕ Exportar e importar

```js
// archivo: saludos.js
export function saludar(nombre) {
  return `Hola, ${nombre}`;
}

// archivo: app.js
import { saludar } from './saludos.js';
console.log(saludar("Luis"));
```
